# Web styles

Global CSS and runtime design tokens for the LunchLineup web app.

## File map

- `README.md` - This folder guide.
- `globals.css` - Tailwind layer imports, CSS variables, shared surface/button/form/workspace classes, public home/document/status health styles, and viewport-constrained responsive shell, scrollable navigation, and scheduler rules.
- `responsive.css` - Standalone scheduling-grid responsive rules retained for reference; this file is not imported by the runtime.
- `tokens.css` - Supplemental token draft retained for design-system reference; runtime styles currently import `globals.css`.
