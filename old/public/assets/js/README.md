# JavaScript Assets

The JavaScript code powers all client-side behaviour.

- **main.js** – Entry point that loads the modules and initialises the app
- **login.js** – UI logic for the login and registration page
- **landing.js** – Scroll-triggered fade-in animations for the landing page
- **dashboard.js** – Company dashboard interactions for company administrators
- **modules/** – Organised ES modules handling state, events and rendering
- **print.js** – Layout snapping and auto-print logic for the printable schedule

See `modules/README.md` for an explanation of each module.
