# Views Directory

Contains reusable PHP/HTML snippets used by the main interface.

- **modals.php** – HTML markup for the various modal dialogs (task editor, staff editor, shift editor, etc.). The file is included near the end of `public/app.php`.
- **print_schedule_view.php** – Template used by `public/api/print_schedule.php` to render the printable schedule with training, tip tracker and chore list cards.
