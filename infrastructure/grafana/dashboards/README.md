# Grafana Dashboards

## Files

- `README.md`: this dashboard folder guide.
- `dashboards.yml`: Grafana dashboard provider configuration.
- `platform-overview.json`: LunchLineup platform overview dashboard, including recent staff-invitation dead-letter transitions, provider-failure, sweep-age, configured-freshness-aware readiness, due/expired/recent-failure backlog, and automatic application-data retention execution age.
- `tenant-deletion-billing.json`: deletion-billing reconciliation dashboard for outcome rates, backlog, oldest-pending age, sweep freshness, and last healthy sweep age.
